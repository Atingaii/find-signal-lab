# Task Archive Index

This directory stores milestone-level task archives for the project.

## Seeded Tasks

- `0001-project-bootstrap`: stage-one skeleton, docs, config, and bootstrap scripts
- `0002-fund-universe-and-data`: V1 universe, master data, daily ingestion, and SQLite warehouse
- `0003-feature-and-label-pipeline`: horizon labels, peer-relative labels, and baseline features
- `0004-baseline-model`: first baseline model and evaluation slices
- `0005-explanation-engine`: deterministic explanation layer
- `0006-dashboard-and-reporting`: API delivery, Streamlit views, and report output

All task folders follow the same structure:

- `README.md`
- `CONTEXT.md`
- `ACCEPTANCE.md`
- `PLAN.md`
- `TODO.md`
- `STATUS.md`
